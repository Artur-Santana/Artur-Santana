  ## Projeto em Destaque

  ### Romma — Sistema de Gestão de Aluguéis Corporativos
  Sistema full-stack que conecta proprietários a locatários de espaços
  comerciais. Gerencia o ciclo completo do contrato: cadastro de unidades,
  contratos com geração automática de parcelas via Edge Function, controle
  de status e painel administrativo.

  `Next.js 16` `Supabase` `PostgreSQL` `RLS` `Edge Functions` `Tailwind CSS v4`
